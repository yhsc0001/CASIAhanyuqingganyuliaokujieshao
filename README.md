# CASIA汉语情感语料库介绍

CASIA汉语情感语料库是一个专门为研究情感语音而设计的语料库。该语料库共包括四个专业发音人，涵盖六种不同的情绪，总计提供了9600句不同的发音。其中，300句为相同文本的发音，100句为不同文本的发音。这些数据可以广泛应用于各种分析实验和研究项目中。

## 数据内容

- **情绪种类**：6种情绪
- **总句数**：9600句
- **相同文本句数**：300句
- **不同文本句数**：100句

## 数据用途
CASIA汉语情感语料库主要用于情感语音的研究和分析。通过该语料库，研究人员可以进行情感识别、语音情感分析、情感语音合成等相关实验和研究。
